# Convert Text App

[Live Demo](https://converttext.vercel.app/)

App com funções para manipulação de textos:

## 1 - Conversor de Textos

- Formato de URL link;
- Formato de frase;
- Palavras capitalizadas;
- Maiúsculas;
- Minúsculas;
- Letras alternadas;
- Inversor de letras

## 2 - Tradutor Binário

Converte textos para binário e vice versa;
